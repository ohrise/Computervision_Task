# 🖼 Image Processing GUI Program

OpenCV + GLUI를 활용하여 구현한 영상 처리 프로그램입니다.  
컬러 및 흑백 영상을 모두 지원하며, 다양한 영상 처리 기능을 GUI 기반으로 수행할 수 있습니다.

---

## 🎯 Assignment Requirements

### ✅ 0. 공통사항

- [x] 컬러 / 흑백 영상 처리 지원
- [x] Read Button을 통한 이미지 불러오기
- [x] Save Button을 통한 결과 이미지 저장

---

## ✏️ 1. Drawing 기능

GLUI 메뉴를 통해 색상 및 선 두께를 조절할 수 있습니다.

- [x] 펜 (Free Drawing)
- [x] 직선 (Line)
- [x] 사각형 (Rectangle)
- [x] 원 / 타원 (Circle / Ellipse)
- [x] 색상 선택 (Color Control)
- [x] 선 두께 조절 (Line Thickness)
- [x] 채우기 (Fill)

---

## ✂️ 2. Cropping

- 마우스로 영역을 선택하여 이미지 일부를 추출
- 선택된 영역을 별도 파일로 저장 가능

---

## 🎚 3. Contrast Adjustment

- [x] Linear Contrast
- [x] Contrast 그래프 출력
- [x] Inverse Transformation
- [x] Gamma Correction

---

## ⚫ 4. Thresholding

- [x] Grayscale Thresholding
- [x] Color Thresholding
- [x] 5가지 Threshold 모드
- [x] Otsu Algorithm
- [x] 임계값 수동 조정
